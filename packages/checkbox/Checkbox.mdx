# Checkbox

Use checkboxes when a user can select one or more options from a list. Checkboxes can be used
individually or grouped under a shared label and help text using `w-checkbox-group`.

## Checkbox group

Use `w-checkbox-group` to add a shared label, help text, optional indicator, and required validation
that enforces at least one selection.

### Form association

Both `w-checkbox` and `w-checkbox-group` are form associated. The group is used for validation only.
Individual checkboxes provide the submitted values.
