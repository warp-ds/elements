Pill

Pill is a type of button that is often used as a filter, but can also be used as a rounded button for overlays, etc.

Props

prop type default
can-close boolean "false"
suggestion boolean false
open-sr-label
string
Label read by screen readers when targeting the pill.
 "Open filter"
close-sr-label
string
Label read by screen readers when targeting the close button.
 "Remove filter {label}"

Events

Name When
w-pill-click Event to be called when the pill is clicked.
w-pill-close Event to be called when the close button is clicked.
